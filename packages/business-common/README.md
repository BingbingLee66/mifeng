# 公共业务资源库

## 介绍

秘锋应用共用业务资源，如
- 常量
- 枚举
- API
- ...

## ⚠️注意

项目目录应以模块划分，如

```bash
├── src
│   ├── mass
│   │   ├── constant.js
│   │   ├── enum.js
│   │   ├── index.js
│   │   └── api.js
│   ├── user
│   │   ├── constant.js
│   │   ├── enum.js
│   │   ├── index.js
│   │   └── api.js
└── index.js

```
